# weather-station
A weather station with ESP32

## Frontend:
- `Vue.js` (Framework)
- `Tailwind CSS` (Style)
- `WebSockets` (Web page to server connection)

## Backend
- `Node.js` (Server)
- `Firebase` (Database)

## Station
- `ESP32` (Microcontroller)
- `DHT22` (Temperature/humidity sensor)

## Hosting
- `GitHub` (Repository and CI/CD)
- `Vercel` (Server hosting)
